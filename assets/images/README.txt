How to add your own images
==========================

1. Drop image files into this folder: assets/images/
2. Make sure each filename matches the "image" field in js/db.js.
3. The game will automatically load them by that filename.

Alternatively, open tool.html in your browser. It will detect missing images
and guide you through downloading each one with the correct filename.

Tips
----
- You can use .jpg, .png, .webp, etc. Just update the extension in js/db.js to match.
- If an image is missing, the game will show a broken image icon.
- To add more assets, use the form in tool.html or edit js/db.js directly.
- After editing js/db.js manually, refresh tool.html to see the changes.
